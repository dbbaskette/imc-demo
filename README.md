# Insurance MegaCorp Demo

A customized implementation of [Diagram Designer](https://github.com/dbbaskette/diagram-designer) for the Insurance MegaCorp demo use case.

## Quick Start

```bash
cp .config.env.template .config.env
# Add your service credentials to .config.env
./deploy-local.sh
```

Access at `http://localhost:8080`

## Diagrams

- **Crash-Processing** - FNOL crash detection and AI-powered claims workflow
- **Telemetry-Processing** - Vehicle telematics data ingestion pipeline

## Documentation

See the [Diagram Designer](https://github.com/dbbaskette/diagram-designer) repository for full documentation on configuration, authentication, and deployment.
